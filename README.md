Commentaires :

Pour réaliser ce projet, on a choisis d’utiliser uniquement du HTML, du CSS et du JavaScript. Le site fonctionne sous forme d’étapes successives (QCM, choix du mouvement, illustration et produits), gérées dans une seule page grâce à l’affichage dynamique contrôlé par JavaScript.

Plusieurs difficultés ont été rencontrées, la vérification des champs du QCM et la synchronisation des données avec l’affichage.
